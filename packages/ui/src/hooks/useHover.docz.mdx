# useHover

`useHover` hook is used to determine if an element is hovered (the cursor is on it).

It returns a `ref` that has to be passed to the element we want to test and a boolean
telling if the element is hovered or not.

### Usage

```jsx
import { useHover } from '@borrow-ui/ui';

export function PageWithElements() {
    // hooks and state
    const [hoverRef, isHovered] = useHover();
    // ....

    // other code
    const hoveredClass = isHovered ? 'link-class__hovered' : '';
    return (
        <div>
            <a ref={hoverRef} className={hoveredClass}>
                Title
            </a>
            <Lorem />
        </div>
    );
}
```
