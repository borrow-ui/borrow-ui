# Link

Links are used to create connections between pages and to external content.
The `Link` component can be used as a standard link (`a` tag)
or customized to use another component (like `react-router-dom`'s `Link`).

## Default tag

The default tag is `a` tag and it is obtained from config function called `getLinkComponent`.
This is a function that returns a valid component or tag.

The setting can be changed with config setter function called `setConfig`.
In your `App.js` or `index.js` file you can call the function and set your custom one:

```jsx
import { Link as RouterLink } from 'react-router-dom';
import { setConfig } from '@borrow-ui/ui';

setConfig('getLinkComponent', () => RouterLink);

export default function App() {
    // ...
}
```

This can always be overridden by specifying the `tag` prop.
