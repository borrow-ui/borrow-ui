# TileLink

`TileLink` component can be used to create home page or section menus.
It uses the `Tile`, `Link` and `Icon` components to generate a clickable Tile.

The `TileLink` component accepts three main props:

-   `icon`: the icon visualized in the tile;
-   `name`: the name of the Tile (larger text);
-   `description`: the description of the Tile (smaller text).

It also accepts a `size` prop, which can be `small` or `large` (default to small).
