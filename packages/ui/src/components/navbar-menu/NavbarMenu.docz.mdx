# NavbarMenu

The `NavbarMenu` component is the ideal way to create menus inside `Navbar`s.
Two components are used underneath: `NavbarMenuTitle` and `NavbarMenuItem`.

`NavbarMenu` takes a list of entry objects (`entries` prop) whose properties are valid `MenuItem` props.
An optional `title` prop will be rendered by using `NavbarTitle` component.
