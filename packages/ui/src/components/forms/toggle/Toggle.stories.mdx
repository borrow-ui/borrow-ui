# Toggle

`Toggle` component can be used as an alternative of checkboxes.

<Preview>
    <Story name="Default">
        <div className="storybook-container borrow-ui">
            <FormsStoryWrapper initialState={{ value: true }}>
                {({ state, setState }) => {
                    return (
                        <div style={{ width: 400 }}>
                            <Field label="Close the window when the process is done">
                                <Toggle
                                    checked={state.value}
                                    onClick={() => setState({ value: !state.value })}
                                />
                            </Field>
                        </div>
                    );
                }}
            </FormsStoryWrapper>
        </div>
    </Story>
</Preview>

Disabled status is supported as well via `disabled` prop:

<Preview>
    <Story name="Toggle disabled">
        <div className="storybook-container borrow-ui">
            <FormsStoryWrapper initialState={{ value: true }}>
                {({ state, setState }) => {
                    return (
                        <div style={{ width: 400 }}>
                            <Field label="Close the window when the process is done">
                                <Toggle
                                    checked={state.value}
                                    disabled={true}
                                    onClick={() => setState({ value: !state.value })}
                                />
                            </Field>
                        </div>
                    );
                }}
            </FormsStoryWrapper>
        </div>
    </Story>
</Preview>
