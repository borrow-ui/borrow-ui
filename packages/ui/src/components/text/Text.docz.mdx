# Text

`Text` component is used to render readable text in a consistent way.
This can be used for posts, long descriptions or the main page content.

By default the used tag is `div`, but it can be overriden by using `tag` prop.

There are also two other sizes available, that can be specified via `size` prop: `small` and `big`.
