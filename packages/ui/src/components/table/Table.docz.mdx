### Search bar

A search bar is available through the `SearchBar` component. This will allow to narrow the list of entries.

To make the search bar visible, set `searchBar` `visible` property to `true`:

<Playground>
    <div className="p-b-20">
        <Table
            searchBar={{ visible: true }}
            columns={[
                { prop: 'id', title: 'ID', width: '100px' },
                { prop: 'name', title: 'Name' },
                { prop: 'group', title: 'Group' },
            ]}
            entries={[
                { id: 1, name: 'first entry', group: 'group 1' },
